# 📂 Dynamic Open Archive

A simple, dynamic archive platform built to help students store, view, and organize resources in one place, packed with a bunch of handy built-in tools.

This is the open-source codebase of the live portal, which is hosted at [n-informatique.top](https://www.n-informatique.top/).


## 🚀 What it does

It's a central portal where you can browse documents and assets, but instead of just being a boring file list, it comes with a complete set of widgets:

*   **Interactive File Explorer**: Easily navigate folders and preview PDF, Word, or other documents right in the browser.
*   **OCR Reader (Text Extractor)**: Can't copy text from a scanned document or slide? There's a built-in scanner to extract text instantly.
*   **Drawing Board & UML Maker**: Sketch quick diagrams or build UML schemas on the fly.
*   **Adaptive Notepad**: Jot down quick notes directly in the app. They save to your browser's local storage automatically.
*   **Study & Break Games**:
    *   *Flashcards*: Create and study cards for quick memorization.
    *   *Mini Games*: Custom Snake game, OctaShot, and Music Destroyer to take a quick study break.
*   **Feedback System**: Users can easily drop comments or request specific files.
*   **Secrets & Easter Eggs**: A couple of hidden surprises if you click around! 😉

## 🛠️ Running it locally

Want to run this yourself? It's just a Next.js app:

1. Clone this repo:
   ```bash
   git clone https://github.com/Brakey-kun/DynamicOpenArchive.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Launch the dev server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) and you're good to go!
